# ⚡ Quick Setup: SQLTools Extension (5 Minutes)

## 🎯 **Goal:** Cursor se Supabase database directly access karna

---

## 📋 **STEP 1: Extension Install (2 Minutes)**

1. **Cursor kholo**
2. **Extensions panel kholo:** `Ctrl + Shift + X`
3. **Search karo:** `SQLTools`
4. **Install karo:** "SQLTools" by Matheus Teixeira
5. **Driver install karo:** "SQLTools Driver - PostgreSQL" (same panel mein search karo)

**Done! ✅**

---

## 📋 **STEP 2: Connection Add (3 Minutes)**

1. **Left sidebar** mein **SQLTools** icon click karo (database icon)
2. **"Add New Connection"** click karo
3. **Connection type:** PostgreSQL select karo
4. **Details fill karo:**

   ```
   Name: Supabase ERP
   Server: db.kexwnurwavszvmlpifsf.supabase.co
   Port: 5432
   Database: postgres
   Username: postgres
   Password: [YOUR_PASSWORD]
   ```

5. **"Test Connection"** click karo
6. **"Save"** click karo

**Done! ✅**

---

## 🔑 **Password Kahan Se Milega?**

**Supabase Dashboard:**
1. https://supabase.com/dashboard/project/kexwnurwavszvmlpifsf
2. **Settings** → **Database** → **Connection string** → **URI** tab
3. Connection string mein `[YOUR-PASSWORD]` wala part copy karo

**Ya phir:**
- Project create karte waqt jo password set kiya, wahi use karo

---

## ✅ **Verify: Kaam Kar Raha Hai?**

1. **SQLTools panel** mein connection expand karo
2. **"Tables"** folder dikhega
3. **Tables expand karo** → Tables dikhengi
4. **Right-click** on table → **"Show Table Records"** → Data dikhega

**Perfect! 🎉**

---
