# 🍃 Sakura Management Portal - Complete ERP System

A comprehensive Progressive Web App (PWA) for Sakura Cafe's complete business management system including Accounts Payable, Raw Material Forecasting, and Warehouse Management.

## 🚀 Features

### 📊 **Accounts Payable Dashboard**
- Complete supplier management
- Invoice tracking and payments
- Purchase order management
- Financial analytics and reporting
- Real-time data from Google Sheets

### 📈 **Raw Material Forecasting**
- Advanced demand forecasting
- Budget planning and analysis
- Inventory optimization
- Cost analysis and savings tracking
- Multi-language support (English/Arabic)

### 🏪 **Warehouse Management**
- Real-time inventory tracking
- Stock level monitoring
- Transfer management
- Purchasing workflow
- Comprehensive reporting

### 🌐 **Progressive Web App (PWA)**
- **Installable** - Add to home screen
- **Offline Support** - Works without internet
- **Mobile Optimized** - Perfect for mobile devices
- **Fast Loading** - Service Worker caching
- **Push Notifications** - Real-time updates
- **Cross-Platform** - Works on all devices

## 📱 Installation

### **Mobile (Android/iOS)**
1. Open the app in your mobile browser
2. Tap the "Install App" button
3. Follow the installation prompts
4. App will be added to your home screen

### **Desktop (Windows/Mac/Linux)**
1. Open the app in Chrome/Edge browser
2. Click the "Install App" button in address bar
3. Follow the installation prompts
4. App will be added to your applications

## 🔧 Technical Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Tailwind CSS, Custom CSS
- **Charts**: Chart.js

- **Fonts**: Cairo (Arabic), Dubai (English)
- **PWA**: Service Worker, Web App Manifest
- **Data Source**: Google Sheets API
- **Hosting**: Surge.sh

## 🛠️ Development

### **Local Development**
```bash

git clone [repository-url]

# Open in browser
open index.html

# Or use local server
python -m http.server 8000
```

### **Deployment**
```bash
# Install Surge
npm install -g surge

# Deploy to Surge
surge . sakura-portal.surge.sh
```
